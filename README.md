# 🏝️ 数学冒险岛

一个爸爸为女儿做的小学数学学习工具。

每一个数学主题做成一个「岛屿」,每一个难度阶梯是一个「关卡」。通过互动 HTML 页面,把抽象概念变成可以拖、可以玩、可以亲眼看见的体验。

## 在线访问

部署在 GitHub Pages: https://wood-zh.github.io/math-island/

## 已上线的岛屿

- **追及岛** — 速度追逐问题(Lv1 经典 / Lv2 时间差 / Lv3 反过来求)

## 怎么用

iPad Safari 直接打开网址,把页面「添加到主屏幕」就像 App 一样。
