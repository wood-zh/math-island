# 标签表

> **不是**预先建好的知识图谱,而是**跟着错题自然长出来的**薄弱地图。
>
> 规则:
> - 转写错题时,如果没有合适的现有标签,**当场新建一行**
> - 标签 ID 用英文小写 + 连字符(例:`fraction-add-sub`)
> - 当某标签的"错题数 ≥ 3"且"关卡状态 = ⬜未开始",主动提议做这个专题
> - "前置依赖"是可选的——只在你转写时确实想到了再填(例:做分数加减前要会通分)

## 关卡状态符号

- ⬜ **累积中**:错题数 < 3,继续观察
- 🟡 **已 3+ 待开发**:可以提议做关卡了
- 🟠 **开发中**:Claude 正在生成
- ✅ **已完成**:岛/关卡上线
- 🔁 **需迭代**:她做完后还有遗留卡点,需要再加一关或改一关
